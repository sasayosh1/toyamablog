🛠️ 新しい管理ツール

📱 記事編集ツール (improved-article-editor.cjs)

コマンドラインから記事を効率的に管理できるツールを作成しました。

// 使用例: // 記事統計情報を表示 SANITY_API_TOKEN="your_token" node improved-article-editor.cjs stats // 記事を検索 SANITY_API_TOKEN="your_token" node improved-article-editor.cjs search "富山" // 記事の全文を表示 SANITY_API_TOKEN="your_token" node improved-article-editor.cjs show "toyama-city-cake-station" // ヘルプを表示 node improved-article-editor.cjs help

主な機能:

  • ✅ 記事の完全な内容表示（分割なし）
  • ✅ 記事検索とフィルタリング
  • ✅ ブロック単位での編集
  • ✅ 統計情報とレポート
  • ✅ バッチ更新機能
